# PinSnippet

Enhance your coding experience with custom snippets in IntelliJ IDEA-based IDEs.

![PinSnippet Icon](path/to/icon.png) <!-- Замените на путь к иконке, если она есть -->

PinSnippet is a plugin for IntelliJ IDEA-based IDEs that allows you to save, manage, and insert custom code snippets efficiently.

## Features

- **Save Snippets**: Save selected code as a snippet with a name and shortcut.
- **Insert Snippets**: Use code completion by typing the shortcut to insert snippets.
- **Manage Snippets**: Organize snippets in a simple JSON file via `Tools > Manage Snippets`.
- **Group Snippets**: Create folders to group your snippets for better organization.

## How to Install

1. Download the plugin ZIP file from the [download page](https://victhegeek.github.io/).
2. In IntelliJ IDEA, navigate to `File > Settings > Plugins` (or `IntelliJ IDEA > Preferences > Plugins` on macOS).
3. Click the gear icon and select `Install Plugin from Disk`.
4. Choose the downloaded `PinSnippet-x.x.x.zip` file and click `OK`.
5. Restart IntelliJ IDEA to activate the plugin.
